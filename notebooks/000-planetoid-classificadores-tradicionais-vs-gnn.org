* Introdução
Este /notebook/ busca demonstrar a superioridade de /Graph Neural Networks/ (=GNNs=) sobre classificadores tradicionais de conteúdo em /datasets/ de grafos de citações /benchmark/, onde a estrutura topológica é reconhecidamente informativa.

Esta é uma prova-de-conceito para embasar o trabalho com o grafo de citações arXiv. Aqui, sabemos que temos um grafo amplamente conexo de artigos (nós) e referências (arestas). Existe mais informação obtível através das /relações entre os artigos/ do que seu próprio texto pode oferecer. Em situações (i.e. nas fases iniciais do trabalho com arXiv) nas quais há poucos elementos conexos no grafo, ou existem poucos nós ou arestas, *a importância infra-textual é maior do que a relacional*. Quando este é o caso, classificadores baseados em TF-IDF produzem melhores resultados do que as /GNNs/. Entretanto, quanto mais informações pudermos *extrair da topologia* deste grafo, melhor se sairão as /GNNs/. É isto que buscamos demonstrar neste /notebook/.

Usamos o dataset *Cora* (e opcionalmente *CiteSeer*) do Planetoid (Yang et al., 2016), nativo do =PyTorch Geometric=. Cora contém 2.708 papers científicos classificados em 7 categorias, conectados por 5.429 citações. As features de cada nó são /bag-of-words/ binárias (1.433 dims).

A comparação é direta e controlada:
- *Baseline*: /Logistic Regression/, /Random Forest/, usam apenas as features textuais (/bag-of-words/), sem ver o grafo.
- *GNNs*: /GCN/, /GraphSAGE/, /GAT/ — usam features textuais *e* estrutura do grafo de citações.

Ambos são avaliados no *mesmo split* (=train/val/test= padrão do Planetoid).

#+begin_src python :results none
from torch_geometric.nn import GCNConv, SAGEConv, GATConv, JumpingKnowledge
import torch.nn as nn


class GCN(torch.nn.Module):
    """GCN com 3 camadas, BatchNorm, residual e Jumping Knowledge."""
    def __init__(self, in_ch, hidden, out_ch, dropout=0.5, num_layers=3):
        super().__init__()
        self.dropout = dropout
        self.input_proj = nn.Linear(in_ch, hidden)
        self.convs = nn.ModuleList([GCNConv(hidden, hidden) for _ in range(num_layers)])
        self.bns = nn.ModuleList([nn.BatchNorm1d(hidden) for _ in range(num_layers)])
        self.jk = JumpingKnowledge(mode='cat')
        self.classifier = nn.Linear(hidden * num_layers, out_ch)

    def forward(self, x, edge_index):
        x = F.dropout(x, p=self.dropout, training=self.training)
        x = F.relu(self.input_proj(x))
        outs = []
        for conv, bn in zip(self.convs, self.bns):
            residual = x
            x = F.relu(bn(conv(x, edge_index))) + residual
            x = F.dropout(x, p=self.dropout, training=self.training)
            outs.append(x)
        return self.classifier(self.jk(outs))


class GraphSAGE(torch.nn.Module):
    """GraphSAGE com 3 camadas, BatchNorm, residual e Jumping Knowledge."""
    def __init__(self, in_ch, hidden, out_ch, dropout=0.5, num_layers=3):
        super().__init__()
        self.dropout = dropout
        self.input_proj = nn.Linear(in_ch, hidden)
        self.convs = nn.ModuleList([SAGEConv(hidden, hidden) for _ in range(num_layers)])
        self.bns = nn.ModuleList([nn.BatchNorm1d(hidden) for _ in range(num_layers)])
        self.jk = JumpingKnowledge(mode='cat')
        self.classifier = nn.Linear(hidden * num_layers, out_ch)

    def forward(self, x, edge_index):
        x = F.dropout(x, p=self.dropout, training=self.training)
        x = F.relu(self.input_proj(x))
        outs = []
        for conv, bn in zip(self.convs, self.bns):
            residual = x
            x = F.relu(bn(conv(x, edge_index))) + residual
            x = F.dropout(x, p=self.dropout, training=self.training)
            outs.append(x)
        return self.classifier(self.jk(outs))


class GAT(torch.nn.Module):
    """GAT com 3 camadas, BatchNorm, residual e Jumping Knowledge."""
    def __init__(self, in_ch, hidden, out_ch, heads=4, dropout=0.5, num_layers=3):
        super().__init__()
        self.dropout = dropout
        self.input_proj = nn.Linear(in_ch, hidden * heads)
        self.convs = nn.ModuleList([
            GATConv(hidden * heads, hidden, heads=heads, concat=True, dropout=dropout)
            for _ in range(num_layers)
        ])
        self.bns = nn.ModuleList([nn.BatchNorm1d(hidden * heads) for _ in range(num_layers)])
        self.jk = JumpingKnowledge(mode='cat')
        self.classifier = nn.Linear(hidden * heads * num_layers, out_ch)

    def forward(self, x, edge_index):
        x = F.dropout(x, p=self.dropout, training=self.training)
        x = F.elu(self.input_proj(x))
        outs = []
        for conv, bn in zip(self.convs, self.bns):
            residual = x
            x = F.elu(bn(conv(x, edge_index))) + residual
            x = F.dropout(x, p=self.dropout, training=self.training)
            outs.append(x)
        return self.classifier(self.jk(outs))

** Treinamento
#+begin_src python :results verbatim
def train_gnn(model, data, epochs=300, lr=0.005, weight_decay=5e-4):
    optimizer = torch.optim.Adam(model.parameters(), lr=lr, weight_decay=weight_decay)
    best_val_acc = 0
    for epoch in range(epochs):
        model.train()
        optimizer.zero_grad()
        out = model(data.x, data.edge_index)
        loss = F.cross_entropy(out[data.train_mask], data.y[data.train_mask])
        loss.backward()
        optimizer.step()
        if epoch % 50 == 0:
            model.eval()
            with torch.no_grad():
                pred = out.argmax(dim=1)
                val_acc = (pred[data.val_mask] == data.y[data.val_mask]).float().mean()
                best_val_acc = max(best_val_acc, val_acc.item())
    model.eval()
    with torch.no_grad():
        pred = model(data.x, data.edge_index).argmax(dim=1)
        test_acc = (pred[data.test_mask] == data.y[data.test_mask]).float().mean().item()
    return test_acc, best_val_acc


NUM_CLASSES = dataset.num_classes
NUM_FEATURES = dataset.num_node_features
HIDDEN = 128
gnn_results = {}

for name, ModelClass, kwargs in [
    ('GCN',       GCN,       {'hidden': HIDDEN, 'dropout': 0.3}),
    ('GraphSAGE', GraphSAGE, {'hidden': HIDDEN, 'dropout': 0.3}),
    ('GAT',       GAT,       {'hidden': 64, 'heads': 4, 'dropout': 0.3}),
]:
    model = ModelClass(NUM_FEATURES, out_ch=NUM_CLASSES, **kwargs).to(DEVICE)
    test_acc, val_acc = train_gnn(model, data)
    gnn_results[name] = test_acc
    print(f"{name:<12} val_acc={val_acc:.4f}  test_acc={test_acc:.4f}")

* Discussão
O dataset Cora demonstra de forma limpa a vantagem das GNNs sobre classificadores de conteúdo:

- /Baselines/ clássicos (sem grafo) atingem ~65-70% de acurácia usando apenas o /bag-of-words/ de cada /paper/.
- /GNNs/ superam consistentemente os /baselines/, chegando a ~83-85%.
- O ganho (~15 p.p.) vem exclusivamente do uso da estrutura do grafo de citações: a /GNN/ agrega representações dos vizinhos, propagando informação sobre a categoria temática através das conexões.

A diferença em relação ao nosso dataset do arXiv é a /densidade e homofilia do grafo/: Cora tem grau médio =~4= e homofilia =~0.81=, enquanto nosso grafo tem grau médio (=0.05= no início, posteriormente =0.5=, =1.8=, talvez =3.4=) e maior mistura entre categorias (especialmente dentro de =cs.*=). Em Cora, o sinal estrutural é mais forte, portanto a vantagem das /GNNs/ é mais pronunciada.
